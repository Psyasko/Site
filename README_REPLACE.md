# Site replacement package

Структура після розбиття:

- `index.html` — структура сторінки.
- `styles.css` — стилі сайту.
- `app.js` — логіка навігації, тестів, історії, PDF-експорту, саморефлексії та практик.
- `tests-data.js`, `explore-data.js`, `soothing-data.js` — дані опитувальників і вправ.
- `assets/` — зображення.

Для заміни репозиторію завантаж увесь вміст цього архіву в корінь GitHub-репозиторію.
