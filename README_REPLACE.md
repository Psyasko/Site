# Site replacement package

Файли підготовлені для повної заміни вмісту репозиторію.

Структура:
- `index.html` — HTML-структура сайту.
- `styles.css` — усі стилі.
- `app.js` — основна логіка сайту.
- `tests-data.js`, `explore-data.js`, `soothing-data.js` — дані тестів, рефлексії та практик.
- `assets/` — зображення.

Для GitHub/Vercel: видали старі файли в репозиторії, завантаж вміст цього архіву, зроби commit.
